# AdventofCode2020
C++ Cmake Repo for Advent of Code 2020

I/O Filepaths are currently set up for the project in VS. The project structure is:

* AdventOfCode/
  * Day_XX/
    * Day_XX.h
    * Day_XX.cpp
    * input.txt
    * CMakeLists.txt
    * etc..
  * Day_XY/
    * ..
  * out/
    * build/
      * Debug/
        * Day_XX/
          * AdventofCodeDayXX.exe/
 *CMakeLists.txt
 
If you are using a differeng layout, you will have to change the filepaths accordingly.
